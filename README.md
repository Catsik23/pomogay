# Помогай (помогай.рф)

Платформа для прямой помощи людям.
Донаты идут напрямую получателю через СБП. Платформа не трогает деньги.

## Технологии
- Python 3.10+
- Flask
- SQLite
- Hugging Face (ИИ-модерация)
- PWA

## Установка и запуск
```bash
pip install -r requirements.txt
python app.py
```

## Статус
Pre-seed. Этап 0 — подготовка инфраструктуры.
